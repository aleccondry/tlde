# SDK Environment for RTOS Emulation

This in-project SDK containerizes Renode and QEMU as emulation frameworks.

> [!NOTE]
> Since in-project SDKs don't support craft parts, to be able to bring the `init_container.sh` script and the Mosquitto configuration, both were added to `hooks/aux` directory.
> So once the workshop is launched with the SDK, the `aux` directory and it's contents get copied to the workshop.
